소개
====
 공통적으로 사용되는 간단한 유틸리티 헬퍼를 모아놓은 라이브러리


사용방법
========
 별도의 (정적/동적) 라이브러리를 만들지 않으므로 프로젝트에 직접 포함해서
 컴파일해야 한다
